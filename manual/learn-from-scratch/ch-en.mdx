# 从汉语到英语

为了让初学者能尽快对英语有初步认识，下面把汉语和英语两种语言的组成做了类比分析，便于快速从已知的概念过渡到未知的概念。

## 语素对照表
语素是指构成语言的要素。

在书写方面，两种语言的构成方式如下：
* 汉语：笔划 > 字 > 词 > 句
* 英语：字母 > 单词 > 句

在发音方面，两种语言的构成方式如下：
* 汉语：声母+韵母+声调 > 单音节 > 字 > 词 > 句
* 英语：辅音+元音 > 单/多音节 > 单词+语调 > 句

| 汉语元素 | 类似度 | 英语元素 |
| :---: | :---: | :---: |
| 笔划 | 高 | [字母](./alphabet) |
| 字/词 | 高 | 单词 |
| 韵母 | 高 | 元音 |
| 声母 | 高 | 辅音 |
| 音节 | 高 | 音节 |
| 声调 | 低 | 语调 |
| 语调 | 高 | 语调 |
